## 🐛 Dépannage

### Les images ne s'affichent pas
- Vérifiez que le dossier `images/` existe
- Vérifiez le nom exact des fichiers image
- Utilisez des chemins relatifs: `images/nom-image.jpg`

### Le formulaire ne fonctionne pas
- Vérifiez la console du navigateur pour les erreurs (F12)
- Assurez-vous que tous les champs requis sont remplis

### Design non responsive
- Effacez le cache du navigateur (Ctrl+Shift+Delete)
- Vérifiez les media queries dans `styles.css`
